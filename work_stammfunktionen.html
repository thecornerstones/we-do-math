Stammfunktionen

Regeln f√ºr das "Aufleiten"

Die Funktion, welche aufgeleitet werden soll wird mit f(x) bezeichnet. Die "aufgeleitete" Funktion (Stammfunktion) wird mit F(x) bezeichnet.

Ganzrationale Funktionen

1) f(x)= xn  F(x)= a/n+1 xn+1 + c

Beispiel:

f(x)= x2  F(x)= 1/3 x3

 

2) f(x)= a * xn  F(x)= a/n+1 * xn+1 + c

Beispiel:

f(x)=2x2  F(x)= 2/3 * x3 + c

 

3) h(x)= f(x) + g(x)  H(x)= F(x) + G(x) + c

 

4) f(x)= (ax+b)n  F(x)= 1/a*(n+1) * (ax+b)n+1 + c

Beispiel:

f(x)= (2x+5)4  F(x)= 1/10 * (2x+5)5 + c

E-Funktion

f(x)= a * ekx  F(x)= a/k * ekx + c

 

Beispiel 1:

f(x)= ex  F(x)= ex + c

Beispiel 2:

f(x)= e2x  F(x)= 1/2*e2x + c

Geometrische Funktionen

f(x)= sin(k*x)  F(x)= -1/k *cos(k*x) + c

 

Beispiel 1:

f(x)= sin(x)  F(x)= -cos(x) + c

Beispiel 2:

f(x)= sin(2x)  F(x)= -1/2 *cos(2x) + c

 

f(x)= cos(k*x)  F(x)= 1/k *sin(k*x) + c

Beispiel 1:

f(x)= cos(x)  F(x)= sin(x) + c

Beispiel 2:

f(x)= cos(2x)  F(x)= 1/2 *sin(k*x) + c
